
# gen

Compositor JSX static site generator

```sh
npm i @compositor/gen
```

```sh
gen . --out-dir dist
```

- Exports static HTML
- Handles multiple routes
- Export to styled-components, glamorous, and more
- Use Lab components in markdown
- Pass data via front-matter
